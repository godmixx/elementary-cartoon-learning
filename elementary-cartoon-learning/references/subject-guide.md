# 学科指南 — Subject Guide

本文档提供小学各学科的知识点组织建议和内容范例，帮助生成贴合课程的学习网页。

## 通用内容结构

每个学习网页遵循统一的内容结构：

1. **情境导入** — 用生活场景或故事引入知识点（1-2 步动画）
2. **核心讲解** — 分 2-3 步讲解知识点本质（配合图示动画）
3. **实例演示** — 用 1-2 个具体例子巩固理解
4. **随堂练习** — 3-5 道题目检测掌握程度
5. **互动游戏** — 拖拽/连线等游戏化巩固
6. **总结回顾** — 知识点要点回顾 + 奖励

## 语文 📚

### 主色调
暖红 `#E74C3C` + 浅黄 `#F9E79F`

### 常见知识点方向

| 年级 | 知识点示例 |
|------|-----------|
| 1-2 年级 | 拼音字母、识字、笔顺笔画、近义词反义词、量词使用 |
| 3-4 年级 | 比喻句/拟人句、成语故事、古诗文理解、段落结构 |
| 5-6 年级 | 修辞手法综合、阅读理解策略、作文结构、文言文入门 |

### 内容建议

- **拼音/识字类**：用图片 + 拼音卡片展示，拖拽匹配拼音和汉字
- **修辞手法类**：对比展示普通句子和修辞句子的效果差异，配动画
- **古诗文类**：逐句讲解诗意，配水墨风格插画（CSS 绘制）
- **成语故事类**：分步展示故事情节，每步配 emoji 场景

### 示例：三年级"比喻句"

```
步骤1: 🌟 "什么是比喻句？" — 展示一个普通句子和一个比喻句，对比效果
步骤2: 🌟 "比喻句的结构" — 本体 + 喻词（像/仿佛/好似）+ 喻体，用高亮标注
步骤3: 🌟 "来试试判断" — 展示 3 个句子，让学生判断哪些是比喻句
测验: 给出句子判断是否比喻句 + 填空补充比喻句
游戏: 拖拽匹配本体和喻体（如：月亮→银盘，星星→宝石）
```

## 数学 🔢

### 主色调
天蓝 `#3498DB` + 浅蓝 `#AED6F1`

### 常见知识点方向

| 年级 | 知识点示例 |
|------|-----------|
| 1-2 年级 | 10/20 以内加减法、认识图形、认识钟表、乘法口诀 |
| 3-4 年级 | 分数的初步认识、面积与周长、两位数乘法、角的认识 |
| 5-6 年级 | 小数运算、方程入门、分数四则运算、圆的面积、百分数 |

### 内容建议

- **计算类**：用实物图（苹果、糖果）演示运算过程，分步动画
- **图形类**：用 CSS/SVG 绘制几何图形，标注边/角/面积，可旋转翻转
- **概念类**：用分割、折叠动画演示分数/小数的意义
- **应用题**：用场景动画还原题目情境，分步列式

### 示例：三年级"分数的初步认识"

```
步骤1: 🍕 "分披萨啦！" — 展示一个披萨被切成 2 份，引出 1/2 的概念
步骤2: 🍕 "切成 4 份" — 同一个披萨切成 4 份，引出 1/4，对比大小
步骤3: 📝 "分数各部分名称" — 分子、分母、分数线，用图示标注
步骤4: ✏️ "读一读写一写" — 练习分数的读法和写法
测验: 比较分数大小 + 看图写分数 + 判断对错
游戏: 拖拽不同分数的图形到对应分数值上
```

## 英语 🔤

### 主色调
紫色 `#9B59B6` + 浅紫 `#E8DAEF`

### 常见知识点方向

| 年级 | 知识点示例 |
|------|-----------|
| 3-4 年级 | 颜色单词、动物单词、数字单词、家庭成员、日常问候 |
| 5-6 年级 | 一般现在时、现在进行时、一般过去时、方位介词、can/can't |

### 内容建议

- **词汇类**：图片 + 单词 + 发音提示（用中文标注近似发音），拖拽匹配图文
- **语法类**：用时间轴/场景动画展示时态变化，对比句型结构
- **对话类**：用卡通角色对话气泡展示日常对话场景
- **游戏类**：单词与图片匹配、填空完成句子

### 示例：三年级"颜色单词"

```
步骤1: 🌈 "Rainbow Colors" — 展示彩虹，引出颜色单词
步骤2: 🔴 "Red, Blue, Yellow" — 逐个展示颜色 + 单词 + 常见物品
步骤3: 🎨 "Let's Practice" — 看图片说颜色单词
测验: 看图选单词 + 单词拼写填空
游戏: 拖拽颜色单词到对应颜色的物品上
```

## 科学 🔬

### 主色调
草绿 `#27AE60` + 浅绿 `#ABEBC6`

### 常见知识点方向

| 年级 | 知识点示例 |
|------|-----------|
| 1-2 年级 | 认识植物、动物分类、天气现象、五官与感觉 |
| 3-4 年级 | 水的三态变化、光的传播、磁铁性质、植物生长 |
| 5-6 年级 | 地球运动、生态系统、简单电路、声音传播 |

### 内容建议

- **自然现象类**：用动画模拟自然过程（水循环、昼夜变化等）
- **实验类**：分步展示实验步骤，配安全提示
- **分类类**：拖拽分类游戏（动物分类、物质分类等）
- **观察类**：放大镜效果展示细节，对比观察

### 示例：四年级"水的三态变化"

```
步骤1: 💧 "水是什么？" — 展示液态水，引出水的基本状态
步骤2: ♨️ "水变蒸气了" — 动画展示加热后水变成水蒸气（蒸发）
步骤3: ❄️ "水变冰了" — 动画展示冷冻后水变成冰（凝固）
步骤4: 🔄 "三态循环" — 展示冰→水→水蒸气→水→冰的循环过程
测验: 判断物态变化类型 + 连线匹配变化过程
游戏: 拖拽不同状态的图片到对应的"固态/液态/气态"区域
```

- **角色扮演类**：通过卡通角色对话展示正确行为

### 示例：二年级"文明礼貌"

```
步骤1: 👋 "打招呼" — 展示小动物互相问好的场景
步骤2: 🙏 "说谢谢" — 展示收到帮助后说谢谢的场景
步骤3: 😊 "说对不起" — 展示不小心碰到别人后道歉的场景
步骤4: 🤝 "文明用语大全" — 汇总常见礼貌用语
测验: 情境选择题（遇到XX情况应该说什么）
游戏: 拖拽礼貌用语到对应的情境图片上
```

| 年级 | 知识点示例 |
|------|-----------|
| 1-2 年级 | 认识颜色、线条形状、简单绘画 |
| 3-4 年级 | 色彩搭配、冷暖色调、对称图形 |
| 5-6 年级 | 透视基础、构图原理、美术流派 |

### 内容建议

- **色彩类**：用色块动画展示颜色混合（红+黄=橙），互动调色
- **形状类**：用 SVG 图形展示几何与有机形状，可拖拽组合
- **构图类**：用对比图展示不同构图效果
- **互动创作类**：提供简单画板功能（Canvas），让学生在线涂色

### 示例：三年级"色彩的三原色"

```
步骤1: 🎨 "三原色是谁？" — 展示红、黄、蓝三个色块
步骤2: 🤔 "混在一起会怎样？" — 动画演示红+黄=橙
步骤3: 🌈 "更多混合" — 演示所有两两混合的结果
步骤4: 📝 "间色和复色" — 介绍间色概念
测验: 判断颜色混合结果 + 选择正确的原色组合
游戏: 拖拽两种原色到混合器，预测结果颜色
```

## 音乐 🎵

### 主色调
金黄 `#FF9800` + 浅金 `#FFE0B2`

### 常见知识点方向

| 年级 | 知识点示例 |
|------|-----------|
| 1-2 年级 | 认识音符、节奏拍打、动物叫声模仿 |
| 3-4 年级 | 简谱入门、乐器认识、节拍强弱 |
| 5-6 年级 | 五线谱基础、音乐术语、乐曲结构 |

### 内容建议

- **音符类**：用动画展示音符时值（全音符=4拍，二分音符=2拍等）
- **乐器类**：用 emoji/SVG 展示乐器图片，配音色描述
- **节奏类**：用视觉节拍器动画，点击跟随节奏
- **简谱类**：展示简谱与唱名的对应关系

### 示例：三年级"认识音符"

```
步骤1: 🎵 "什么是音符？" — 展示音符是记录声音长短的符号
步骤2: 🥁 "拍一拍" — 全音符(4拍) vs 二分音符(2拍) vs 四分音符(1拍)
步骤3: 🎼 "简谱里怎么写" — 展示简谱中的音符表示
步骤4: ✋ "一起来拍节奏" — 互动节拍器，跟随点击
测验: 根据节拍数选择对应音符 + 判断音符时值
游戏: 拖拽音符到对应的拍数区域（1拍/2拍/4拍）
```

## 内容难度适配原则

| 年级 | 语言风格 | 举例方式 | 交互提示 |
|------|---------|---------|---------|
| 1-2 年级 | 极简单句，大量拼音辅助 | 用食物/玩具/动物举例 | 每步都有明确语音/文字提示 |
| 3-4 年级 | 简单句为主，可含少量成语 | 用生活场景/学校情境举例 | 关键步骤有提示 |
| 5-6 年级 | 可使用适当学术用语 | 用抽象概念+具体例子结合 | 最少提示，鼓励自主探索 |
